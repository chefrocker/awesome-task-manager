# Awesome Task Manager

A powerful, visual task management plugin for Obsidian.
Ein leistungsstarkes, visuelles Aufgabenverwaltungs-Plugin für Obsidian.

---

## 🇬🇧 English

### Features

- **Visual Dashboard** with 4 tabs (Today, This Week, Open, All)
- **Inline Editing** for quick changes directly in the table
- **Detail View** for full task editing with images and notes
- **Smart Automations** (auto status, auto completion date, color coding)
- **Tags** with autocomplete and learning suggestions
- **Excel Export** with one click (including Excel formulas)
- **Recurring Tasks** (daily, weekly, monthly, yearly, custom)
- **Reminders** with desktop notifications
- **Image Support** with date stamps and auto-rename on completion
- **Multi-language** (English, German – extensible)
- **No Dependencies** – works standalone, no other plugins needed
- **Mobile Support** – works on Obsidian Mobile (iOS/Android)

### Installation

1. Open Obsidian Settings → Community Plugins → Browse
2. Search for "Awesome Task Manager"
3. Click Install, then Enable

## 🇩🇪 Deutsch

### Funktionen

- **Visuelles Dashboard** mit 4 Tabs (Heute, Woche, Offen, Alle)
- **Inline-Bearbeitung** für schnelle Änderungen in der Tabelle
- **Detail-Ansicht** für vollständige Bearbeitung mit Bildern und Notizen
- **Intelligente Automatismen** (Auto-Status, Auto-Datum, Farbcodierung)
- **Tags** mit Autocomplete und lernenden Vorschlägen
- **Excel-Export** mit einem Klick (inklusive Excel-Formeln)
- **Wiederkehrende Aufgaben** (täglich, wöchentlich, monatlich, jährlich)
- **Erinnerungen** mit Desktop-Benachrichtigungen
- **Bilder** mit Datumsstempel und Umbenennung bei Abschluss
- **Mehrsprachig** (Deutsch, Englisch – erweiterbar)
- **Keine Abhängigkeiten** – funktioniert eigenständig
- **Mobile Unterstützung** – läuft auf Obsidian Mobile (iOS/Android)

### Installation

1. Öffne Obsidian Einstellungen → Community Plugins → Durchsuchen
2. Suche nach "Awesome Task Manager"
3. Klicke Installieren, dann Aktivieren
